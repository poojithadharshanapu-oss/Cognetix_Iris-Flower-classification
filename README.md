# Iris Flower Classification

## Project Overview
This machine learning project predicts Iris flower species (Setosa, Versicolor, Virginica) using petal and sepal measurements.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Model Used
- K-Nearest Neighbors (KNN)

## Results
- Achieved high classification accuracy on the Iris dataset.
- Visualized feature relationships using correlation heatmaps.
- Evaluated model performance using confusion matrix.

## Dataset
Iris Dataset from Scikit-learn/UCI Repository.
